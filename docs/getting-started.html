Getting Started

Install TSM and run your first analysis in under two minutes.

Prerequisites

  • Rust 1.70+ (install via rustup.rs)
  • TypeScript source files (.ts or .tsx)

Installation

Build from source

bash
git clone https://github.com/gabrielrf97/tsm.git
cd tsm
cargo build --release
# Binary at target/release/ts-static-analyzer

Install globally

bash
cargo install --path .

First Analysis

Point TSM at a TypeScript file or directory:

bash
# Analyze a single file
ts-static-analyzer analyze src/utils.ts

# Analyze an entire directory (recursive)
ts-static-analyzer analyze src/

# Analyze multiple paths
ts-static-analyzer analyze src/ lib/ types/

Understanding the output

ColumnDescription
FilePath to the TypeScript source file
FunctionFunction or method name (<anonymous> for unnamed)
Line1-based line number where the function is declared
LOCLines of Code (including blank and comment lines)
SLOCSource Lines of Code (blank and comment lines excluded)
ComplexityCyclomatic Complexity — color-coded green/yellow/red
NestingMaximum nesting depth
ParamsNumber of formal parameters

Filtering results

bash
# Only show functions with CC ≥ 10
ts-static-analyzer analyze src/ --min-complexity 10

# Only show functions with ≥ 50 LOC
ts-static-analyzer analyze src/ --min-loc 50

# Show verbose output (skipped files reported to stderr)
ts-static-analyzer analyze src/ --verbose

# Show timing information
ts-static-analyzer analyze src/ --timing
