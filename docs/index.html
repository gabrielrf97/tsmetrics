What is TSM?

TSM (TypeScript Static Analyzer) is a command-line tool that parses TypeScript and TSX files using tree-sitter and computes a comprehensive set of software quality metrics. It is written in Rust and designed for speed, correctness, and scriptability.

TSM computes metrics at three levels:

  • Function-level — Cyclomatic Complexity, Halstead Volume, Maintainability Index, LOC, nesting depth, parameter count
  • Class-level — WMC, DIT, NOM/NOAM/NOOM, WOC, TCC, CBO, RFC, NOI
  • File-level — Technical Debt, SLOC
  • React — JSX nesting depth, number of used components (NUC)

It also detects four object-oriented design smells inspired by Object-Oriented Metrics in Practice (Lanza & Marinescu): God Class, Brain Method, Feature Envy, and Refused Bequest.

Features

⚡

Fast

Files processed in parallel using Rayon. Handles large codebases without breaking a sweat.

🎯

Accurate

Built on tree-sitter's full AST—no regex heuristics. Handles generics, decorators, and TSX.

⚙️

Configurable

Override warning/error thresholds per metric in a tsm.yaml config file.

📤

Multiple Outputs

Table, JSON, CSV, and self-contained HTML report formats.

Quick Start

# Install from source
cargo install --path .

# Analyze a directory
ts-static-analyzer analyze ./src

# JSON output
ts-static-analyzer analyze ./src --format json
